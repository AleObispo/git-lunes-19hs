COMANDOS BASICOS:

cd (check directory)
nos permite navegar entre los directoprios de la computadoras (carpetas) utilizando diferencters varienates.

cd .. => vuelve al direcvtori anterior
cd <folde> => abre el directorio especificado
cd \ => abre el directorio raiz (c:)

mkdir (make direcotry)
	permite crear directorios con los nombres que especificamos despues de comando

"mkdir <folder> <folde>"
